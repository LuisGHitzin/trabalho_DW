Explicando e exemplificando o uso das seguinte tags:

Tag table

Cria uma tabela bidimensional composta por linhas e colunas de células contendo dados.
Como por exemplo:

A1 B1 C1
A2 B2 C3

Tag del

A del tag define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.
Como por exemplo: Essa "palavra"" foi excluída

Tag ins

Tag usada para definir um texto que foi inserido ou alterado.
Exemplo: Essa "palavra"" foi inserida

Tag sub

Tag usada para definir um texto subscrito.
Exemplo: fórmula do H20

Tag sup

Tag usada para definir um texto subrescrito.
Exemplo: a2 + b2 = c2

Tag blockquote

Tag usada para definir uma citação externa.
Exemplo:

Tag q

Tag usada para definir uma citação simples.
Exemplo:

Tag abbr

Tag usada para definir uma abreviação. Importante para fornecer informações úteis para navegadores, sistema de tradução e mecanismos de busca.
Exemplo:

A Sigla - IFPR

Tag address

Tag usada para definir as informações de contato do autor do documento. Se estiver dentro de uma tag article irá definir as informações do autor do artigo. A tag pode incluir o endereço físico, URL, e-mail, telefone, mídia social, coordenadas geográficas e etc.
Exemplo:

site IFPR. Acesse irati.ifpr.edu.br/

Tag cite

Tag usada para definir o título de uma obra (livro, música, filme, pintura, escultura ou etc.
Exemplo:
